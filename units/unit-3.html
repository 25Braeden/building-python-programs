Lesson 4: Loops in Python

Loops let you execute a block of code repeatedly, which is useful when you want to perform repetitive tasks efficiently. Python supports two primary types of loops: for loops and while loops. Let's explore each in detail.

For Loops

The for loop iterates over a sequence of items, such as elements in a list, characters in a string, or numbers generated by the range() function. This means the loop will run once for each item in the sequence. 

for i in range(5):
    print(i)

Explanation:

  • range(5) generates a sequence of numbers from 0 up to (but not including) 5: 0, 1, 2, 3, 4.
  • The variable i takes each value from this sequence one by one.
  • print(i) outputs the current value of i during each iteration.

This loop will print the numbers 0 through 4, each on a new line.

While Loops

The while loop runs as long as a certain condition remains true. This makes it great for loops where you don't know in advance how many times it will run, but you want it to continue until some condition changes. 

x = 0
while x < 5:
    print(x)
    x += 1

Explanation:

  • We start with x = 0.
  • The condition x < 5 is checked before each iteration.
  • If true, the loop executes the indented code: print x and then increase x by 1 (x += 1).
  • Once x reaches 5, the condition becomes false, and the loop stops.

This loop also prints 0 through 4, just like the for loop example.

Loop Control Statements

Sometimes you want to change the behavior of loops by exiting early or skipping parts of the loop. Python provides two important statements for this:

  • break: Immediately stops the loop, even if the loop condition is still true.
  • continue: Skips the rest of the current loop iteration and moves to the next one.

Example using break:

for i in range(10):
    if i == 5:
        break
    print(i)

Explanation: This loop prints numbers starting from 0. When i becomes 5, break stops the loop entirely, so it prints 0 to 4 only.

Example using continue:

for i in range(5):
    if i == 2:
        continue
    print(i)

Explanation: When i is 2, the continue statement skips the print(i) for that iteration. So the output is 0, 1, 3, 4 (skipping 2).

Nested Loops

You can put loops inside other loops, called nested loops. This is useful when you want to work with multiple dimensions of data, such as rows and columns. 

for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

Explanation:

  • The outer loop runs 3 times with i values 0, 1, 2.
  • For each i, the inner loop runs 2 times with j values 0 and 1.
  • The print statement shows the current values of i and j.
  • This results in 6 lines printed (3 × 2).
