Lesson 4: Loops in Python

Loops are a fundamental concept in programming that let you repeat a block of code multiple times, saving you from writing repetitive code manually. In Python, there are two primary types of loops: for loops and while loops. Each serves a unique purpose depending on the situation.

For Loops

A for loop is used to iterate over a sequence, such as a list, string, or range of numbers. It automatically goes through each item in the sequence, executing the code inside the loop for each item. This is useful when you know beforehand how many times you want to loop.

Example: Print numbers from 0 to 4 using range().

for i in range(5):
    print(i)

Here, range(5) generates a sequence of numbers from 0 up to, but not including, 5. The variable i takes each value from this sequence one at a time, and the loop body prints it. So this loop prints 0, 1, 2, 3, and 4.

While Loops

A while loop repeats as long as a specified condition remains true. You control when the loop stops by changing values within the loop that affect the condition. This type of loop is useful when you don't know exactly how many times the loop should run in advance.

Example: Print numbers from 0 to 4 using a while loop.

x = 0
while x < 5:
    print(x)
    x += 1

In this example, the loop continues while x is less than 5. Each time through the loop, it prints the current value of x, then increments x by 1 with x += 1. When x reaches 5, the condition x < 5 becomes false, and the loop stops.

Loop Control Statements

Python provides two special statements to control the flow inside loops:

  • break: Immediately stops the loop, skipping any remaining iterations.
  • continue: Skips the rest of the current loop iteration and moves on to the next one.

Example of break to exit a loop early:

for i in range(10):
    if i == 5:
        break  # Exit loop when i is 5
    print(i)

This loop prints numbers 0 through 4. When i equals 5, the break statement stops the loop immediately.

Example of continue to skip an iteration:

for i in range(5):
    if i == 2:
        continue  # Skip printing when i is 2
    print(i)

This loop prints 0, 1, 3, and 4. When i is 2, the continue statement skips the print statement and goes directly to the next iteration.

Nested Loops

You can place loops inside other loops. These are called nested loops. The inner loop completes all its iterations for every single iteration of the outer loop.

Example: Nested for loops printing pairs of i and j values.

for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

In this example, the inner loop runs twice for each value of i. So it prints all combinations of i (0 to 2) and j (0 to 1).
