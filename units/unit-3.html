Lesson 4: Loops in Python

Loops are a fundamental concept in programming that let you repeat a block of code multiple times without writing it over and over. In Python, you have for loops (for iterating sequences) and while loops (running until a condition is false).

For Loops

Example: Print numbers 0 to 4 with for and range(). Copy/paste all the examples below so you can see their outputs.

for i in range(5):
    print(i)

While Loops

Example: Print numbers 0 to 4 with a while loop.

x = 0
while x < 5:
    print(x)
    x += 1

Loop Control

Use break and continue to adjust flow:

for i in range(10):
    if i == 5:
        break
    print(i)

for j in range(5):
    if j == 2:
        continue
    print(j)

Nested Loops

for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")
