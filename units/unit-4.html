Lesson 5: Functions in Python

Functions are reusable blocks of code designed to perform a specific task. They help organize programs into manageable pieces, making your code cleaner, easier to read, and maintain.

Defining a Function

Use the def keyword followed by the function name and parentheses. Inside the parentheses, you can specify parameters (inputs). The function body contains the code to run. 

def greet():
    print("Hello, world!")

Calling a Function

Once defined, you call (execute) the function by writing its name followed by parentheses: 

greet()

Functions with Parameters

Functions can accept inputs, called parameters, which allow you to pass different values each time you call the function. 

def greet(name):
    print("Hello, " + name + "!")

greet("Alice")
greet("Bob")

Return Values

Functions can return a value back to the caller using the return keyword. This lets you use the result elsewhere in your program. 

def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # Output: 8

Common Function Patterns

def is_even(number):
    return number % 2 == 0

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

Why Use Functions?

  • Code reuse: write once, use multiple times.
  • Modularity: break complex problems into smaller parts.
  • Readability: make your code easier to understand.
  • Maintenance: fix or update one place instead of many.
