Lesson 3: Control Flow in Python

Control flow lets you make decisions and execute different blocks of code based on conditions. Python uses if, elif, and else statements along with Boolean expressions.

if Statements

x = 10
if x > 5:
    print("x is greater than 5")

if–else

if x % 2 == 0:
    print("even")
else:
    print("odd")

if–elif–else

if x > 0:
    print("positive")
elif x < 0:
    print("negative")
else:
    print("zero")

Logical Operators

age = 20
if age >= 18 and age < 21:
    print("adult but under 21")

if not (age < 0):
    print("age is non-negative")

Nested Conditions

num = 7
if num > 0:
    if num % 2 == 1:
        print("positive odd")
    else:
        print("positive even")
