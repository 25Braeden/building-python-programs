Lesson 6: Lists and Loops

Lists are a fundamental way to store multiple values in one variable. Loops allow you to automate repetitive tasks, especially useful when working with collections like lists.

Creating a List

Lists use square brackets [], with items separated by commas.

fruits = ["apple", "banana", "cherry"]
print(fruits)

Accessing Elements

You can index into a list (zero-based):

print(fruits[0])  # apple
print(fruits[-1])  # cherry

Modifying Lists

fruits[1] = "blueberry"
fruits.append("orange")
print(fruits)

Looping Over Lists with for

for fruit in fruits:
    print(fruit)

Looping with while

i = 0
while i < len(fruits):
    print(fruits[i])
    i += 1

Common List Methods

  • append(item)
  • remove(item)
  • insert(index, item)
  • pop()
  • len(list)

Why Use Lists and Loops?

  • Store multiple values easily.
  • Automate repetitive tasks with loops.
  • Build more complex programs by combining these features.
