Lesson 6: Lists and Loops

Lists are a fundamental way to store multiple values in one variable. Loops let you automate repetitive tasks, especially useful when working with collections like lists.

Creating a List

Lists are created using square brackets [], with items separated by commas. 

fruits = ["apple", "banana", "cherry"]
  print(fruits)

Accessing Elements

Lists are ordered, and you can access items by their index (starting at 0). 

print(fruits[0])  # Output: apple
  print(fruits[-1])  # Output: cherry (last item)

Modifying Lists

You can change list contents by assigning new values or using list methods:

fruits[1] = "blueberry"
  fruits.append("orange")
  print(fruits)

Looping Over Lists with for

Use a for loop to iterate over each item in a list: 

for fruit in fruits:
      print(fruit)

Looping with while

You can also use a while loop with an index to iterate through a list: 

i = 0
  while i < len(fruits):
      print(fruits[i])
      i += 1

Common List Methods

  • append(item): Adds an item to the end.
  • remove(item): Removes the first occurrence of the item.
  • insert(index, item): Inserts an item at a specific position.
  • pop(): Removes and returns the last item.
  • len(list): Returns the number of items.
fruits.append("kiwi")
  fruits.remove("apple")
  print(fruits)

Why Use Lists and Loops?

  • Store multiple values easily in one place.
  • Automate repetitive tasks with loops.
  • Work with collections of data, such as names, numbers, or objects.
  • Build more complex programs by combining these features.
