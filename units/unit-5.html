Lesson 6: Lists and Loops

Lists are a fundamental way to store multiple values in one variable. Loops allow you to automate repetitive tasks, especially useful when working with collections like lists.

Creating a List

Lists use square brackets [], with comma-separated items.

fruits = ["apple", "banana", "cherry"]
print(fruits)

Accessing Elements

print(fruits[0])  # apple
print(fruits[-1])  # cherry

Modifying Lists

fruits[1] = "blueberry"
fruits.append("orange")
print(fruits)

Looping with for

for fruit in fruits:
    print(fruit)

Looping with while

i = 0
while i < len(fruits):
    print(fruits[i])
    i += 1
