Unit 0: Python Basics

Introduction to Python

Welcome to the Python Basics lesson! In this unit, you'll learn some of the fundamental building blocks of Python programming.

Variables

A variable in Python is used to store data. Think of it as a container that holds a value. For example: 

my_number = 10
my_string = "Hello, Python!"
is_active = True

Data Types

Python has several basic data types:

  • Integers: Whole numbers (e.g., 10, -5).
  • Floats: Decimal numbers (e.g., 3.14, -0.001).
  • Strings: Text (e.g., "Hello", 'world').
  • Booleans: Logical values (True or False).

The Print Function

Use the print() function to display information. For example: 

print("Hello, World!")
print(my_number)
print("Is active:", is_active)
