Unit 0: Python Basics

Introduction to Python

Welcome to the Python Basics lesson! In this unit, you'll learn the fundamentals of Python programming.

Variables

A variable in Python is like a container to store data. For example: 

my_number = 10
my_string = "Hello, Python!"
is_active = True

Data Types

Basic data types include:

  • Integers – e.g., 10, -5
  • Floats – e.g., 3.14
  • Strings – e.g., "Hello"
  • Booleans – True or False

The Print Function

Use the print() function to display output:

print("Hello, World!")
print(my_number)
