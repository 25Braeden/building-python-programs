Lesson 1: Welcome to Python

Python is a powerful yet simple high-level programming language that is loved by developers and data scientists alike. It is known for its simplicity and readability, making it a great choice for beginners.

Real-World Uses of Python

Despite Python's simplicity, it's a tool that powers many exciting real-world applications! Here are some things you can eventually do with Python:

  • Create Video Games: Design and develop your own games using libraries like Pygame.
  • Build Websites and Apps: Use frameworks like Django and Flask to create interactive websites and mobile apps.
  • Analyze Data: Dive into data science and explore trends, patterns, and statistics.
  • Automate Tasks: Write scripts to handle repetitive tasks and make life easier.
  • Explore Robotics and AI: Bring machines to life with robotics and artificial intelligence projects.

As you learn more, you'll discover even more exciting possibilities. But today, we start with the basics—using print() to talk to your computer.

Focus on print()

In this lesson, we will learn all about the print() function. It's one of the most fundamental functions in Python and is used to display output on your screen.

Your First Python Program

Traditionally, the first program you write in any language prints "Hello, World!" to the screen. Try running the following code (it should already be loaded into the code editor): 

print("Hello, World!")

Look at that! You've just printed your first message.

More print() Examples

The print() function can do more than just display text. It can perform arithmetic and even combine text. Enter these examples into the code editor and run them: 


# Arithmetic example: adding two numbers
print(5 + 5)  # Expected output: 10

# String concatenation: joining two strings
print("Hello, " + "Python!")  # Expected output: Hello, Python!

# Combining text and numbers
print("My age is", 25)  # Expected output: My age is 25
print("Her name is", "Alice") # Expected output: Her name is Alice

Great job! Each time you run your code, you’re learning a little more about how to communicate with your computer.
