Lesson 2: Variables in Python

Variables are containers for storing data values. In Python, you don’t need to declare a type—just assign a value using the = operator.

Assigning Variables

For example:

age = 25
name = "Alice"
print("Age:", age)
print("Name:", name)

Here, age holds the integer 25 and name holds the string "Alice."

Understanding Variable Types

Although Python is dynamically typed (you don’t need to explicitly declare a variable’s type), it’s important to know the common types you’ll be working with:

  • Integers (int): Whole numbers without a decimal. For example: 42 or -3.
  • Floating Point Numbers (float): Numbers with a decimal point. For example: 3.14 or -0.001.
  • Strings (str): Sequences of characters enclosed in quotes. For example: "Hello, World!".
  • Booleans (bool): Represent truth values, either True or False.

You can check a variable’s type with the type() function: 

num = 10
print(type(num))  # Output: <class 'int'>
pi = 3.14
print(type(pi))   # Output: <class 'float'>
text = "Hello"
print(type(text)) # Output: <class 'str'>
is_student = True
print(type(is_student))  # Output: <class 'bool'>

Using Variables in Expressions

You can perform arithmetic operations with numbers and concatenate strings: 

length = 10
width = 5
area = length * width
print("Area:", area)
greeting = "Hello, " + name + "!"
print(greeting)

User Input and Variables

Get input from the user using input(): 

user_input = input("Enter your favorite color: ")
print("Your favorite color is", user_input)
