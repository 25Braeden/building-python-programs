Unit 1: Python Programming

1.1 Introduction to Python

Python is a versatile programming language known for its readability and simplicity. In this unit, we'll cover the basics of Python, including its syntax and core concepts.

1.2 Variables and Data Types

Variables allow you to store and manipulate data. Python is dynamically typed, meaning you don’t have to declare a variable’s type explicitly. 


# Example: Assigning values to variables in Python
name = "Alice"
age = 30
print(f"{name} is {age} years old.")

1.3 Control Flow

Control flow statements enable your program to make decisions. Below is an example of a conditional statement in Python: 


# Example: Using a conditional statement in Python
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")
