Lesson 2: Variables in Python

Variables are containers for storing data values. In Python, you don’t need to declare a type—just assign a value using the = operator.

Assigning Variables

For example:

age = 25
name = "Alice"
print("Age:", age)
print("Name:", name)

Here, age holds the integer 25 and name holds the string "Alice."

Understanding Variable Types

Common types include:

  • int: Whole numbers, e.g. 42.
  • float: Decimal numbers, e.g. 3.14.
  • str: Text in quotes, e.g. "Hello".
  • bool: True or False.
num = 10
print(type(num))     # <class 'int'>
pi = 3.14
print(type(pi))      # <class 'float'>
text = "Hello"
print(type(text))    # <class 'str'>
is_student = True
print(type(is_student))  # <class 'bool'>

Using Variables in Expressions

length = 10
width = 5
area = length * width
print("Area:", area)

greeting = "Hello, " + name + "!"
print(greeting)

User Input

user_input = input("Enter your favorite color: ")
print("Your favorite color is", user_input)
